** 무기체계와 미래 기술 발전 방향 - 위성체계와 발전 방향**
- 무기 개발에 이어 양산으로 넘어감
- 시장성이 매우 좋은게 우주 분야이기 때문에 관심을 많이 가지면 좋겠음
- LIG D&A의 목표는 "Space Total Solution"(GEO-MEO-LEO)을 목표를 하고 있음 (궤도운용-LEOP/IOT-위성체 제조-위성체 개발-지상체 개발/운영)
  * 연구소와의 협업을 또 중요하게 생각하고 있음

[기본]
- 2만대 이상의 위성이 돌고 있음
- 우주 항공청에 산업화를 하자고 노력하고 있으나, 위성 사업에으로는 돈이 되지 않음
- 위성같은 경우에는 우리나라 특성상 꼭 자국 개발을 해야하는데 돈이 많이 들어 정부의 수주를 받아 진행하고 있음
- 우주가 뭔지부터 알아야 함
  * Space : 도달 가능한 우주(인간의 목표)  /  * Universe : 관측 가능한 우주  /  * Cosmos : 철학적인 개념을 포함하는 우주
  * Satellite(위성) - Natural : ex) 달  /  Artificial : 인공
- 위성을 이해하기 위해 꼭 알아야 할 핵심 용어
  * 우주 : 지구 해수면 기준 100km 이상 떨어진 지점부터
  ** 우주 어디에도 지구의 영향을 받지 않는 곳은 없음. 유효의 영향을 말하는 거임
  * 인공위성(Artificial satellite) : 인위적으로 궤도에 올려 놓은 물체
- 궤도 - 고도
  * LEO(Low Earth Orbit) : 저궤도(200~2000km)
  * MEO(Medium Earth Orbit) : 중궤도(2000~35786km) -> 방사능때문에 잘 활용하지 않지만, 저궤도에 우주 쓰레기가 너무 많아 활용할 수는 없는지 고민하는 곳
  * GEO(Geostationary Earth Orbit) : 정지궤도(약 35786km) -> 지구 자전 주기와 물려있어 멈춰있는 걸로 보임 ex) 항법위성
  * HEO(Highly Elliptical Orbit) : 장타원 궤도(A:40000, P:1000km) -> 타원인데 엄청 길게 돌아 가장 가까워졌다가 가장 멀어지기도 함(비용이 크고, 수명이 좋지 않음)
- 궤도 - 주기
  * GSO(GeoSynchronous Orbit) : 지구동기궤도 -> 지구 자전 주기와 동일
  * semi-Synchronous orbit : 지구반동기궤도
  * super-Synchronous orbit : 지구초동기 궤도
- Polar Orbit : 극궤도 -> 신호를 빨리빨리 받을 수 있음(ex. 북극에 지상 수신 안테나가 많음)
- SSO(Sun Synchronous orbit) : 태양동기궤도 -> 관측 위성에 많이 쓰임 (특정 지역 타켓으로 하면 하루에 두번 관측 가능)
- 위성 개발 모델 구분(하우징 차이가 있음)
  * 1대 만드는데 정지궤도 기준 6천억가량 들어감. 저궤도는 2~300억. -> 어떻게 싸게 만들까에 대한 고민을 하고 있음
  * 개발단계별 모델  /  용도  /  비고
  ** DM(Development Modle) : 핵심기술 검증 -> 요즘에는 많이 사용하지 않음
  ** EM(Engineering Modle, 공학모델) : 전기적 기능 및 성능, 인터페이스 확인 목적 -> ETB(Engineering Test Bed)에 활용
  ** QM(Qualification Modle) : 환경시험을 통한 인증용 모델, 비행 모델로 사용 제한 -> ETB or 인터페이스 시험(FM 확인용도)
  ** PFM(Proto-Flight Modle) : 검증된 제품을 재 제작할 경우 -> RM 대비 시험레벨 높음. 잘 발생하지 않음
  ** FM(Flight Modle, 비행모델)
  * (FPGA 같은 경우에는 2~400만원 정도) 시험을 많이 거침
- 위성 시스템 구성 - 궤도와 자세가 중요함
  * 1) 위성체(Space Segment) : 본체(Bus) + 탑재체(Payload) -> 본체와 탑재체 모두 중요함
  * 2) 지상체(Ground Segment) : 지상국, 운영시스템
  * 3) 발사체(Launch Segment) : 로켓, 발사 시스템
  * 4) OD(Orbit Determination, 궤도 결정)
  * 5) AD(Attitude Determination, 자세 결정)
- 저궤도 위성(500~50km) : 다목적실용위성 6호(아리랑 6호, KOrea Multi-Purpose Satellite-6), 긴급대응 군집 큐드 위성(EO/IR Cube(18U) Satellite), 초소형 SAR 위성, LIG 초소형 SAR 위성(자체 개발)
  * SAR 위성(흑백) : 레이다 신호로 영상을 만드는 장비로 광학 카메라는 Passive(태양이 있어야 함)인데, 이건 레이다의 특정을 그대로 따라가 기후에 강함
  * 저궤도에서 주로 하는 것은 통신, 관측과 광학 위성을 준비하고 있음
- 정지궤도 위성 : 정지궤도공공복합통신위성(천리안위성 3호, GK3), 한국형위성항법시스템(KPS, KOrean Positioning System), 정지궤도 기상-우주기상 위성(천리안위성 5호, GK5)
  * KPS 위성은 정확도를 cm 급으로 높여보자는 취지로 개발하게 됨
  * GK5는 기상 위성으로 현재 기상 예보는 GK2를 사용해 하고 있는데 더 정확도를 높여보자는 취지로 개발하게 됨

- 연구 개발, 양산, 운영유지

[해양무기체계 소개]
- 해군 수중 무기체계 전분야(ASW(Anti-Submarine Warfare, 대잠전), MCM(Mine Counter Measrue, 대기뢰전) 등) 개발에 참여
  * Sonar/C2 System : 수상함 소나체계/전투체계, 수중감시체계, 해양무인체계 통합지휘통제
  * Guided Weapon System : 중어뢰, 경어뢰, TACM(자항식기만기, 부유식기만기)
  * Autonomous Vehicle System : AUV(무인잠수정, 기뢰/감시 탐색용)
- 1990년 초 중어뢰(백상어) 체계개발로 시작해 다양한 해군 무기체계로 지속 확대/발전
  * 수상함 소나, 잠수함 소나, 감시체계, 수중유도무기, 기만기, 해양무인체계(AUV/MUM-T, C2)
- 수중 환경의 특이점 : 수중 음향신호의 경우 수온/해저지형/퇴적물 특성에 따라 매우 복잡한 특성 보유
  * RADAR(RAdio Detection And Ranging) vs SONAR(SOund Navigation And Ranging)
  ** 수중에는 다양한 소리가 있기 때문에 특정 소리를 잡아내는 소나가 중요한 기술임
- 소나(SONAR, SOund Navigation And Ranging)
  * 수동소나(수동모드 어뢰 탐지) : 적잠수함에서 어뢰발사 - 어뢰방사소음 발생 - 어뢰방사소음 전송 - 어뢰신호 수신
  * 능동소나(능동모드 잠수함 탐지) : HMS에서 능동 핑 송신 - 능동 핑 전송 - 적잠수함에서 핑반사 - 반사 핑 신호 수신
  * 소나체계 운용 흐름 : 신호수신 - 수신신호 처리 및 전송 - 신호처리 - 탐지화면 제공
- 수상함 소나체계
  * 운용 목적 : 적 잠수함 탐지, 식별, 추적
  * 운용 개념 : 능/수동 소나 운용
  * 음파발생기를 이용한 음파 발신, 표적함 반사음 신호 수신, 수신신호 탐지/분석/식별/표적 좌표 추적
  * ex) 장애물 회피 소나, 디핑소나, 선체 고정형 소나, 가변심도 소나(수온이 다른걸 고려), 예인 선배열 소나, 어뢰음향대항체계
- 소나 개발 현황
  * FF(좌우구분빔형성) -> FFG(능동소나기술) -> KDX-1,2,3 -> 광개토-3 B-2
- 잠수함 소나체계
  * 운용 목적 : 적 수상함/잠수함 탐지/식별/추적, 수중통신
  * 운용 개념 : 수동소나 운용(99%), 능동소나 운용(1%, 표적함 발생 음향 신호 수신, 수신신호 탐지/분석/식별 표적 좌표 추적)
  ** 능동소나는 회피 등의 목표로 사용되고 있음
  * 수동선측 배열
- 임무 장비 중심의 사업으로 패러다임이 많이 바뀌었음
- 해저 고정형 소나체계
  * 운용 목적 : 항만감시체계 (수중침투 표적 탐지/식별/추적 및 경보가 가능한 감시 시스템 구축)
  * 운용 개념 : 원거리 지향성 수동소나, 중거리 음향/자기 복합소나, 근거리 고주파 능동소나
  * 표적함 발생 음향 신호 수신, 표적함 발생 자기 신호 수신 분석, 고주파 음파 송수신을 통한 수영자/소형 침투정 탐지
- 소노부이(Sonobuoy) : 작은 1회용 소나를 막 투입하여 관측이 되면 초계기 투입하고 어뢰 투입
- 어뢰 : 수중유도무기
- 카이텐 : 자폭어뢰(제2차 세계대전)
- 정밀 유도는 얼마나 끝까지 정확하게 타격하는지가 중요
- 유선유도와 항적탐지는 최첨단 기술이라고 볼 수 있음
  * 유선유도 -> 대음향 대항 -> 항적탐지(기포항적 탐지) -> 초공동어뢰 -> AI 적용(임무계획, 최적경로, 표적 식별)
- 해군이 훈련하는 것 중 하나가 수중환경 데이터 수집하는 것
- 잠수함에서 쏘는 어뢰는 중어뢰 수동 소나 -> 상대 함정이 회피 기동을 하기 때문에
- 요격어뢰(기만기 쏘고 다 해도 결국엔 타격하는)랑 초공동어뢰(물속에서 총알처럼 날라가는, 앞을 진공상태로 만드는)가 없음 -> 다른 나라도 거의 없음
- 궁극적인 건 완전한 무인화
- 어디로 가는지 모르기 때문에 통신이 매우 중요
- 중어뢰(Heavy weight Torpedo)
  * 운용 목적 : 잠수함에 탑재하여 적 수상함 및 잠수함 공격을 목표로 함
  * 운용 개념 : 능/수동 복합 탐지, 항적 탐지
  * 6m정도 되는 백상어와 범상어는 탑재해야하는 곳이 동일하기 때문에 둘레나 크기가 유사
  * 근사에서 찢어버리는 용도
- 경어뢰(Light weight Torpedo)
  * 운용 목적 : 수상함/헬기/항공기 등에 탑재하여 적 잠수함 공격을 목표로 함
  * 운용 개념 : 능/수동 복합 탐지
  * 타격만 하면 되어서 지향성 탄두 사용
- 수중무기체계 발전 방향 : 대상 표적체계 발전추세에 따라 작전영역 확대, SALVO, 장거리, 고속추진이 가능한 무기체계로 진화 중
- 기뢰 (바다속의 지뢰)
  * 운용 목적 : 접근/접촉 배들를 격침, 설치 지역 해상 봉쇄
  * 운용 개념 : 자율주행(또는 수동)으로 해당 지역 이동 및 매설, 접근/접촉 표적선 격침
  * 수중 자율주행 및 매설(자항기뢰), 기뢰매설함을 이용한 수동 매설(기뢰)
- 기만기(Decoy)
  * 운용 목적 : 아군 수상함/잠수함의 생존성 확보를 위한 적 어뢰 기만
  * 운용 개념 : 적의 어뢰를 기만하기 위함
- 무인잠수정
  * 운용 목적 : 감시정찰, 기뢰 매설/탐지/제거 등의 수중 임수 무인 수행
  * 운용 개념 : 지정된 경로로 임무 수행, 모선과 주기적 통신(수중 통신, 해상 통신)
  * 수중 잠항 시 수중통신, 부상시 해상 통신을 통한 데이터 송신 및 명령 수신
  * 소나, 카메라, 열영상 등 다수의 센서 운용(임무센서부, 전지부, 자율주행 통제부, 자울주행 센서부, 추진후부)
  * 군집 임무제어 및 통합기술, 수중 네트워크 기술, 저주파 SAS 기술
- 어뢰 1발에 3~40억 정도함
- 전쟁형태변화 : 고가 유인 플랫폼에서 저가/다수형 무인 자폭체계로 바뀜
- 3D 프린팅을 사용하여 저가형으로 빠르게 생산 효율성을 높인게 확인되었음
- 핵어뢰는 안에 핵을 가지고 있어 항만에 터트려 쓰나미를 일으켜 주변을 쓸어버리는 것도 있음
- 무인기에 어뢰를 탑재하는 핵심 기술 단계는 접어들었음
- 광개토-3 통합소나체계
  * 저주파 선체 고정형 능동탐기로 Direct Path 및 Bottom Bounce 탐지 / 가변심도 능동음탐기 및 다기능선배열로 저소음 표적 CZ 탐지
  * 대잠전술운용, 무장통제(홍상어, 청상어, 자항식/부유식 기만기)
  * 소노부이, 대잠헬기, 무인수상정과 협동 작전 수행
- 무인잠수정용 대잠 탐지소나
- 중어뢰-2(범상어)  /  경어뢰-2(성능개량 경어뢰 2발 SALVO 운용)  /  자항기뢰  /  CIWS-2(사격통제체계, AESA, EO/IR, 함포 --> 1초에 수십발을 쏴서 드론에도 효과적임)
- 원거리탐지용 음향센서 : 수동소나, 양상태 능동소나 신호처리/정보처리 기술 개발 중
- 항만감시체계  /  해군음향정보관리체계(NAIMS-2, 해군이 수집한 데이터 분석)

** 무기체계와 미래 기술 발전 방향 - 무인체계 사업의 이해 및 발전 방향 **
- 무인체계가 주도하는 전장에서의 승패는 결국 첨단 기술력으로 귀결
- 우버 공유서비스 개념의 Defense Cloud와 AI 기술, 저궤도 위성통신을 접목하여 실시간 전투 수행개념 정립 중
- 작전의 경제성, 효율성, 속도 향상을 위한 무인체계, 리플리케이터 구상

- 무인체계 = 시스템의 집합(System of System) ex. 드론, UGV, USV
  * 구성요소 : 센서(EO/IR, Radar), 제어기(MCU/SoC), 통신(데이터링크), AI/알고리즘, 지휘통제, 무장/임무수행
- 무인체계에서의 임베디드 역할
  * 센서=데이터 수집, RTOS=실시간 처리, 통신=데이터 안정성, 제어=오류 방지, 안전=Fail-safe
- 모든 것이 무력화 됐을 때 통신 등이 되게 만드는 역할을 해야함
- 대기업=체계통합, 중견기업=핵심모듈, 중소기업=SW/센서, 개발자=성능결정
- 다양한 플랫폼에서 무인 또는 유/무인 복합운용(MUM-T)
- 지상무인체계 개발 현황
  * 초소형 생체모방 로봇용 SW 프레임워크 개발
  * 군사용 곤충형 지상이동로봇 기술 (도심 작전 지원을 위한 생체모방 영상 센서, 이동 매커니즘, MEMS 기반 힘 센서를 적용한 군사용 곤충형 지상 로봇)
  * 생체조류 형상/비행 모방
  * 지상 소형 감시 정찰 로봇 (PUGV, 지상 소형 감시 정찰 로봇) -> 험지 이동을 위한 원격 통제 기술, 자율주행, 공통플랫폼화 기술 기반
  * 초소형 로봇 군집운용 통제기술 -> 핵심 표적에 대한 감시/정찰/추적 임무 시 최적의 임무 효과를 달성하기 위해 다수 로봇의 자율 임무계획/군집 운용 기술 개발
  * VILS 기반 군 운용환경 AI 학습 기술
  * AI 전술참모 및 군집 운용 -> AI 전술참모 기술 및 MR기반 군집 지휘통제 기술 개발
  * 유/무인복합체계 공통 아키텍처 및 프레임워크 -> 유/무인복합체계의 모듈과/공통화를 통한 상호운용성과 확장성 증대
  * 4족 보행 로봇(Ghost Robotics) -> 유무인 복합 체계 협업 구성을 위한 4족 보행 로봇
- 해양무인체계 개발 현황
  * 함상 진회수(내리고 올리는) 기술 -> 고내항/고속 플랫폼 설계/운용 기술 및 EOTS, 무장장치, 항재밍 모듈 등 탑재장치 모듈화 기술
  * 탑재무장 개발현황(자체개발) -> 무인수상정 탑재무장 기술 RCWS(원격사격통제체계, Remote Controlled Weapon Station)
  * 군집 자폭형 무인수상정 개발 -> 타격 후 자폭을 위한 비궁 발사 모듈 장착 가능 군집 자폭형 무인수상성 개발 중
  * 수중자율기회탐색체 -> 대기뢰전 능력을 보강하기 위해 수중 자율운행으로 기뢰탐색, 감시 및 정찰 수행
  * 무인잠수정 군집운용 -> 기뢰대항작전 능력 향상을 위해 수중 군집 운용, 수중 네트워크, 고해상도 데이터 획득 센서, AI 기반 기뢰 탐지 기술 개발
  * 해양 수중보행로봇 -> 해저 환경에서 자율 보행/유영하며, 스스로 임무를 관리/계획/수행할 수 있도록 하는 자율 이동 및 임무 제어 기술 개발
- 공중무인체계 개발 현황
  * 전술 군집 무인기 자율 임무 재계획 기술
  * M&S(해군교전급모델)
- AI기반 MUM-T 통합지휘통제체계
  * 초연결 네트워크(C4I 체계 연동 기술 etc.), 유/무인복합체계, 자율화/AI, 클라우드 환경
- "여러분의 코드 한 줄이 전장의 결과를 바꿀 수 있습니다."

** 무기체계와 미래 기술 발전 방향 - 감시정찰체계와 발전 방향 **
- 10대 국방 무기체계 : 지휘통제통신 무기체계, 감시/정찰 무기체계, 기동, 함정, 항공, 화력, 방호, 사이버, 우주
- 감시정찰 : 적을 관찰하여 변화하는 움직임부터 지형, 날씨 등 필요한 자료를 알아내기 위해 관찰/탐색하는 활동
- 레이다가 필요한 이유
  * 전천후성(RF 전파는 구름/안개/비/야간을 투과해 기상 조건에 무관하게 24시간 탐지 유지), 거리 정밀도, 침투성
  * 무기체계의 눈(eyes)
  * 얼마나 멀리 보는지, 얼마나 잘 구분하는지, 얼마나 정확히 재는지, 얼마나 자주 갱신하는지가 중요
- 대포병탐지레이더 : 적 포탄/로켓의 비행궤적을 탐지해 발사원점과 탄착지점을 역추적/산출하는 레이더
  * 대화력전에서 '포병의 눈' 역할
- 장거리레이다(400m) : 한국방공식별구역(KADIZ) 내 항공기탐지/추적을 24시간 수행, 포착 정보를 공군 중앙방공통제소(MCRC)로 실시간 전송하여 방공작전을 지원하는 무기체계
  * 멀리가야하기 때문에 산 꼭대기에 있음
  * 순수 국내기술로만 개발, 26년 양산 및 전력화
- FA-50용 AESA 레이다 : 국내최초 전투기용 공랭식 항공기용 AESA 레이다
  * 열이 엄청 많은데 공랭식으로 하며 우월한 성능을 보유
  * 기존 FA-50 기계식레이다를 대체하며 소형/경량 및 최신기술 적용으로 공군 작전능력 강화기여
  * 순수 LIG 자체 기술, 국산화율 약 95% 이상 -> 안의 반도체는 미국산을 쓰고 있음. SW는 다 국산
  * 유지보수측면까지 고려하였을 때 기존 레이다 대비 약 75% 수준의 저렴
- 레이다는 '임무중심'으로 최적화 : 같은 항공기더라도 '원거리 감시/지휘', '사격통제', '해상표적 식별', '기상회피', '정밀영상획득' 중 무엇이 핵심인지에 따라 안테나 형상, 운용주파수 대역, 빔 운용방식, 해상도 및 탐지거리 균형이 달라짐
  * 탐색레이다, 추적레이다, 사격통제레이다(무장 교전 지원), 감시정찰레이다(광역 ISR), SAR 레이다(고해상도 영상 형성), ISAR 레이다(선박 형상 식별), 기상레이다
- L-Band/S-Band : 지상용/해상용 레이더, 탄도탄 조기경보 레이더, 장거리 레이더 -> 안테나 큼
- C-Band / X-Band(대부분)
- 환경은 항상 변화하고 이에 따라 레이다는 진화하고 있음
  * 주요 환경의 변화 : 표적의 다양화, 반도체의 등장, 현대 전장환경, AI기반 신호처리
- 1세대-기계식스캔(고고도 대형 항공기, 실리콘), 2세대-PESA(저고도 침투기/순항미사일, 비소갈륨), 3세대-AESA(스텔스기/정밀유도무기, 질화갈륨(GaN)), 차세대-AI-AESA(UAV 군집/극초음속 표적, Diamond/SiC)
- 현대 전자전 위협 유형 : 전파 방해(Jamming), 수동 탐지(ESM/ELINT), 신호 기만(Spoofing)
